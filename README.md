# 微擎简介

#### 同步自码云<https://git.oschina.net/we7coreteam/pros>的稳定版本,每月最少更新一次;微擎是一款非常强大的微信管理平台,全面支持第三方开发,拥有模块应用商店,可以购买到自己任意需求功能,以及定制;微擎是开源软件,商业公司使用需购买授权,具体去<http://www.we7.cc>了解详情.
